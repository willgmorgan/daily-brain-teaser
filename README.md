## How to Install on Your Phone

### iPhone (iOS)
1. Open Safari and navigate to the app URL
2. Tap the **Share** button (square with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Name it "Brain Teaser" and tap **Add**
5. The app icon will appear on your home screen!

## Hosting the App

To make this available to your friends, you need to host it online. Here are the easiest options:

### Option 1: GitHub Pages (Free)
1. Create a GitHub account
2. Create a new repository called "brain-teaser"
3. Upload all the files:
   - daily-brain-teaser.html
   - manifest.json
   - service-worker.js
   - icon-192.png
   - icon-512.png
4. Go to Settings > Pages
5. Enable GitHub Pages from the main branch
6. Your app will be live at: `https://yourusername.github.io/brain-teaser/daily-brain-teaser.html`

### Option 2: Netlify (Free)
1. Go to netlify.com
2. Drag and drop the folder with all files
3. Get instant URL like: `https://your-brain-teaser.netlify.app`

### Option 3: Vercel (Free)
1. Go to vercel.com
2. Import your GitHub repo or upload files
3. Deploy instantly
